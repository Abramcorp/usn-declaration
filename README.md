# Налоговая декларация ИП УСН 6% — мастер в 4 шага

Локальное приложение: загружаете банковскую выписку — получаете заполненную декларацию по форме КНД 1152017.

---

## 🚀 Как запустить — самый короткий путь

### Windows (двойной клик)

1. Откройте папку проекта в Проводнике.
2. **Дважды кликните `ЗАПУСТИТЬ.bat`** ← вот эта кнопка.
3. При первом запуске подождите 1–2 минуты, пока установятся зависимости.
4. Браузер откроется автоматически на `http://localhost:8000`.

### macOS / Linux / или если `.bat` не работает

Откройте терминал в папке проекта и выполните:

```bash
python3 run.py
```

Или на Windows в PowerShell / cmd:

```bat
py run.py
```

Скрипт `run.py` сам:
- проверит и при необходимости установит библиотеки из `requirements.txt`,
- создаст папки `data/` и `uploads/`,
- запустит сервер и откроет вкладку в браузере.

**Чтобы остановить** — закройте окно терминала или нажмите `Ctrl+C`.

## 📋 Требования

- Python **3.10 или новее** ([скачать](https://www.python.org/downloads/))
- При установке на Windows обязательно поставьте галочку «**Add Python to PATH**»
- Для конвертации в PDF: **LibreOffice** ([скачать](https://www.libreoffice.org/download/)) — должен быть в PATH (`soffice`)

Если LibreOffice не установлен — Excel-версия декларации всё равно будет сгенерирована, а PDF придётся сохранить вручную из Excel.

## 🧙 Как работает мастер (4 шага)

1. **Загрузка выписки** — выбираете год и перетаскиваете `.txt` 1С-выписку из банка.
   ИНН и ФИО подтянутся автоматически. Указываете ОКТМО и код ИФНС.
   Опционально: загружаете `.xlsx` с чеками ОФД (если были наличные продажи).

2. **Выручка по кварталам** — видите банк, ОФД (только наличные) и итог по 4 кварталам
   + нарастающим итогом. Логика дедупликации: безналичные чеки ОФД игнорируются,
   т. к. они уже попадают в банк как эквайринг.

3. **Персонал и взносы** — отвечаете «есть ли работники» (управляет лимитом 50 % vs 100 %)
   и вводите уплаченные взносы нарастающим итогом. Приложение рассчитает налог к уплате
   по каждому периоду.

4. **Декларация** — выбираете период (квартал / полугодие / 9 мес / год), дату подачи
   и получаете заполненную декларацию в PDF и XLSX.

## 🛠 Troubleshooting

| Проблема                                    | Решение                                                   |
|---------------------------------------------|-----------------------------------------------------------|
| «python не является внутренней командой»    | Установите Python 3.10+ и поставьте галочку «Add to PATH» |
| `.bat` мгновенно закрывается                | Запустите `py run.py` из открытого cmd — увидите ошибку   |
| PDF не создаётся                            | Установите LibreOffice, убедитесь что `soffice` в PATH    |
| «ИНН не извлекся из выписки»                | Файл не в формате 1С ClientBank — попросите у банка `.txt` в 1С-формате |
| Браузер не открылся                         | Откройте вручную `http://localhost:8000`                  |
